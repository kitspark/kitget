# kitget

kitget is a one-shot source slicer for the Kitspark ecosystem.

It clones or reads a local copy of `kitspark/core`, copies a strict subset
of directories, and materializes them into a variant repository as owned source.

## What kitget is

- A one-time code injector
- A filesystem slicer
- A development convenience

## What kitget is not

- A dependency manager
- A runtime library
- An updater or sync tool

## Usage

```bash
kitget
